## Сайт логопеда-дефектолога (лендинг)

Лендинг для Елены Ивановны Мушкаревой: услуги, документы, этапы, стоимость и форма заявки.

### Как запустить локально

- **Самый простой способ**: открыть `index.html` двойным кликом.
- **Правильнее (как на хостинге)**: поднять локальный сервер.

Если установлен Python:

```bash
python -m http.server 5173
```

Потом открыть в браузере: `http://localhost:5173`

### Структура

- `index.html` — разметка страницы
- `styles.css` — стили
- `main.js` — интерактив (меню, FAQ, анимации, форма)
- `photo.jpg`, `diploma.jpg`, `cert-speech.jpg`, `test.jpg` — изображения

### Форма заявки

Форма не отправляет данные на сервер (чтобы не хранить персональные данные).
После отправки:

- открывается WhatsApp с готовым текстом заявки
- текст заявки дополнительно копируется в буфер обмена (если браузер разрешает)

Номер WhatsApp можно поменять в `main.js` в объекте `CONTACTS`.
